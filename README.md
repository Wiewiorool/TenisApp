## TENIS RESERVATION

This project is an automation bot built with Java and Selenium that automatically books tennis courts on the Tenis4U
platform.

The application logs into the user account, monitors court availability in real time, and performs a reservation as soon
as the desired date and time become available.

## Tech Stack

* Java
* Selenium
* Gradle

## HOW TO RUN

Clone the repository

Build the project using Gradle:

./gradlew build
Run the main class
